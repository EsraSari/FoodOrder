📖 Proje Açıklaması

Bu proje, .Net ve Entity Framework kullanılarak geliştirilmiş kapsamlı bir sistemdir. Sistem, ürünler, menüler, kullanıcı kayıtları, girişleri, siparişler ve müşteri bilgilerini yönetmek için kullanılır.

🚀 Özellikler

  • Müşteri Bilgileri Girişi: Müşteriler adres ve iletişim bilgilerini girebilirler (Ülke, il, ilçe, adres detayı, e-posta, telefon).
  • Sipariş Yönetimi: Siparişleri listeleme ve yeni sipariş girişi (Tek ürün veya ürün+menü seçimi).
  • Ürün Yönetimi: Yeni ürün girişi ve mevcut ürünleri yönetme.
  • Menü Yönetimi: Daha önce girilmiş ürünlerle yeni menü oluşturma.

🛠️ Kurulum Adımları

  1. Depoyu klonlayın: 
    git clone https://github.com/username/repo.git
  2. Proje dizinine gidin: 
    cd repo
  3. Bağımlılıkları yükleyin: 
    dotnet restore
  4. Veritabanını güncelleyin: 
    dotnet ef database update
  5. Projeyi çalıştırın: 
    dotnet run

🍰 Katkıda Bulunma

Bu projeye katkıda bulunmak isterseniz, lütfen depoyu fork edin ve bir özellik dalı kullanın. Pull request'ler sıcak karşılanır.

  1. Depoyu fork edin.
  2. Yeni bir dal oluşturun: 
    git checkout -b ozellik/ozellik-adi
  3. Değişikliklerinizi yapın ve commit edin: 
    git commit -m 'Bir özellik ekle'
  4. Dala gönderin: 
    git push origin ozellik/ozellik-adi
  5. Bir pull request oluşturun.
